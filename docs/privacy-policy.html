Arcanum Exam — Privacy Policy

Last updated: March 26, 2026

1. Overview

Arcanum Exam ("the Game") is developed by Kingzi ("the Developer"). This policy explains what data the Game accesses and how it is handled.

2. Data We Access

When you launch the Game as a Discord Activity, the Game requests the following through Discord's OAuth2 system:

— Your Discord username and display name (via the identify scope), used to show your name to other players during a game session.

The Game does not access your email address, friend list, messages, or any other Discord data.

3. Data We Store

The Game does not store any personal data persistently. All game state exists only in server memory for the duration of a game session and is discarded when the session ends. There are no user accounts, no databases, and no analytics or tracking systems.

4. Data Shared with Third Parties

The Game does not share any data with third parties. There are no ads, no analytics services, and no external tracking.

5. Cookies and Local Storage

The Game may use your browser's session storage to enable reconnection if you temporarily lose connection during a game. This data is limited to a session token and room identifier, and is automatically cleared when you close your browser.

6. Hosting

The Game is hosted on Render (render.com). When you use the Game, your connection passes through Discord's proxy infrastructure and then to Render's servers. Please refer to Render's Privacy Policy and Discord's Privacy Policy for information about how those services handle data.

7. Children's Privacy

The Game does not knowingly collect personal information from children. Access to the Game requires a Discord account, which is subject to Discord's own age requirements.

8. Changes to This Policy

The Developer may update this policy at any time. Changes will be reflected by an updated date at the top of this page.

9. Contact

For questions about this policy, you can reach the Developer via GitHub: github.com/ChurchmanKingzi
